# Syntax

```json
{
    key:value,
    key:value,
    ...
}

[ele1,ele2,ele3]

[{},{},{}]
```

* 数据在key:value对中
* 数据由逗号 , 分隔
* 使用斜杆 \ 来转义字符
* 大括号 {} 保存对象
* 中括号 [] 保存数组，数组可以包含多个对象



key: 

- 字符串

value:

- 字符串（string）`"str"`

- 数值(number) : 整数/浮点数

- true,false, null

- 对象（object）  `{, }` 访问:` obj.feather`

- 数组（array）  ` [, ]` 访问: `array[ord]`
